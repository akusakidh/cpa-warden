## 📝 What is cpa-warden?

cpa-warden is a simple command-line tool designed to help you scan, check, and manage CPA (Cost-Per-Action) accounts. It works in popular CPA management setups by keeping track of your account inventory, probing usage, and assisting with maintenance tasks. You do not need technical experience to use it.

This tool runs on Windows and uses Python and SQLite behind the scenes to organize and analyze your CPA authorization data.

## 💻 System Requirements

Before you start, make sure your computer meets these minimum requirements:

- Operating System: Windows 10 or later
- Processor: Any modern CPU (Intel or AMD)
- Memory: At least 2 GB RAM available
- Disk Space: 100 MB free space for the program and data
- Internet connection to download and update the tool

You do not need to install anything else, as the package contains everything it requires.

## 🔧 Features

cpa-warden offers:

- Inventory scanning: Quickly find and list CPA authorization records on your system.
- Usage probing: Check how each account is being used or accessed.
- Account maintenance: Update account statuses, fix common issues, or clean stale records.
- A simple, interactive interface that guides you step-by-step.
- Data storage using SQLite for local records.
- Works fully offline after download.
